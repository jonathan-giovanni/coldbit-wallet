# ColdBit Wallet — Bitcoin Cold Storage App

ColdBit Wallet is a fully offline, privacy-focused Bitcoin cold wallet designed for absolute security and user sovereignty.  
It performs all critical wallet operations locally on the device — including key generation, transaction signing, and address derivation — without connecting to any third-party servers, analytics systems, or external APIs.

The wallet adheres strictly to Bitcoin Improvement Proposals (BIPs) standards such as **BIP32**, **BIP39**, and **BIP44**, ensuring deterministic and interoperable key management.  
It uses the system’s **Secure Enclave (iOS)** or **StrongBox/Keystore (Android)** for secure private key storage, combined with biometric authentication and an 8-digit PIN for double-layer protection.

ColdBit Wallet embodies the philosophy of minimalism, anonymity, and full control — a cold wallet designed for those who demand sovereignty over convenience.
